AI Research Copilot

AI Research Copilot is an AI-powered tool that automatically generates structured academic papers from a user-provided topic. It produces Abstract, Introduction, Literature Review, Methodology, Results, and Conclusion, fetches scholarly references from CrossRef and Semantic Scholar, and formats them in APA style. The tool also generates a PDF of the paper and a BibTeX file for references.

The user-friendly web interface is built with Gradio, allowing users to select the number of references, enable AI model options, and download their paper instantly. Ideal for researchers, students, and hackathon projects, it streamlines academic writing and literature review processes.

Features:

AI-generated academic sections using Groq and Hugging Face LLMs

Automatic retrieval and deduplication of scholarly references

APA citation formatting and BibTeX generation

PDF export of the complete paper

Modern, intuitive web UI for fast paper generation

Tech Stack: Python, Gradio, Requests, Nest-Asyncio, Pybtex, ReportLab

Use Case: Quickly prototype academic papers, generate literature reviews, and assist in research projects.
